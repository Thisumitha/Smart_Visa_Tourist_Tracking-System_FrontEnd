# Smart Visa Tourist Tracking System - Frontend

This is the frontend application for the Smart Visa Tourist Tracking System.

## Technologies Used
- React 19
- Vite
- TailwindCSS
- Chart.js
- Axios
- React Router DOM
- SweetAlert2

## Getting Started

1. Clone the repository.
2. Install dependencies: `npm install`
3. Run the development server: `npm run dev`
4. Access the application at the URL provided by Vite.

## Build for Production
To build the application for production, run: `npm run build`
